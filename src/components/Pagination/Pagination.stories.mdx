# Pagination

## 標準コンポーネント

前ページ、次ページなど複数のページに遷移する際にコンテンツ下部に設置する。

PC 表示時、以下のボタンバリエーションが全て含まれる状態では、前後 3 ページ個表示とする。表示領域にナビゲーションが全て収まらない場合は、モバイル表示時の状態へと変化する。

※ページネーション操作が、入力フィールドとなる場合については、今後定義予定。（検索結果などにおいて表示件数が多い場合ユーザーに任意に入力させる）

### ボタン配置

配置は左から下記の順に配置する。

最初へ < 前へ < 省略表記 < ページ No ボタン Before < カレントページ表記 < ページ No ボタン After < 省略表記 < 次へ < 最後へ

### カレントページ表記

カレントページ No を表示します。

### 省略記号

総ページ数がページ No ボタンの範囲より多い場合に描写される。

前 3 ページより、若いページナンバーがある場合は、ページ No ボタン Before の前に表示されます。

前 3 ページより、年長のページナンバーがある場合は、ページ No ボタン After の後に表示されます。

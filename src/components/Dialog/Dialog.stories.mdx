# Dialog

ダイアログは、重要な情報を知らせ、決定や選択を求めるコンポーネントです。複数のタスクを含む場合があります。一時的にユーザーの動作を中断する必要がある場合のみに使用し、多用を控えてください。

## ダイアログを閉じる操作

ダイアログを閉じる操作は、ダイアログの種類により異なります。

### 「キャンセル」ボタン

ダイアログの操作を中止する

### 「閉じる」ボタン

ダイアログを閉じる

## ライティング

理解性を高めるために、ダイアログの中のテキストは短く論理的にデザインする必要があります。否定表現の重複は避け、「〜する」などの不要な文章は省きましょう。

- 問いの中の動詞と同意行動が一致している
- 「キャンセルのキャンセル」否定表現が重複しておりユーザーの混乱を招く
- タイトル・本文が簡潔ではない、不要な言葉「〜する」「〜しない」が冗長

# Button

## 標準コンポーネント

カラーは各省庁共通とし、個別のブランドカラーには左右されないものとします。

### プライマリ

プライマリボタンは、ページ内で最も重要なアクション事に使用します。（例：「同意」、「送信」、「応募」）

### セカンダリ

プライマリボタンに比べて優先度が下がるアクションに対して使用します。（例：「問い合わせ」）

### ターシャリ

プライマリボタン及びセカンダリボタンと比べて優先度が低い 3 つ目のアクションに対して使用します。（例：「キャンセル」）

## マージン

各省庁を横断して閲覧した際に、ロゴのばらつきが出ない様に 24px 以下でのロゴの設置は避けて下さい。

配置エリアはモバイル及びタブレットの h.40 x w.140px（青エリア）とし、エリアの中心線とロゴの中心線を揃え左端のグリッドに合わせて下さい。

## ターゲット領域

ターゲットは、指のタッチやマウスのようなポインティングデバイスによるポインタアクションを受け入れる領域を指します。
